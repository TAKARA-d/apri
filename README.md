# パワプロ風ペナントモード（簡易再現）

1時間で遊べることを重視した、CLI版の「ペナントモード」再現です。

## できること

- 6球団のリーグ戦をシーズン通して進行
- 日程ごとに試合を自動シミュレーション
- 勝敗・勝率・ゲーム差の順位表表示
- チーム運営コマンド
  - **特訓**: 打撃/投手力を恒久強化
  - **休養**: 疲労回復
- シーズン終了で優勝決定

## 実行

```bash
python3 pennant.py
```

## 操作

- `n`: 次の日程を進める
- `a`: シーズンを最後まで自動進行
- `s`: 順位表を表示
- `r`: ロスター状態（能力・疲労）表示
- `t`: 特訓（打撃 or 投手）
- `e`: 休養
- `q`: 終了

## 設計メモ

- 「パワプロの雰囲気」を優先し、能力値で得点期待値が変わる軽量モデルを採用。
- 乱数シード固定で同じ展開を再現可能。
- 実データではなく、オリジナル球団・簡略ルールです。

## テスト

python3 -m unittest -v
