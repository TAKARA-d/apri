# APRI ペナントマネージャー 2026

本格的に遊べることを目標にした、**キャラクターデザイン付き Web ペナント運営ゲーム**です。

## 主な要素

- 6球団リーグの長期シーズン（総当たり＋ホーム/アウェイ）
- 選手18名×6球団のロスター管理
- 選手の能力値（打撃/投手/体力/カリスマ）とコンディション
- 試合シミュレーション（攻守戦力・士気・ホーム補正）
- 特訓/休養/スカウト獲得/予算管理
- SVGベースの簡易キャラクターデザイン（全選手の顔グラ表示）
- セーブ/ロード（localStorage）

## 起動方法

依存パッケージ不要です。静的ファイルを配信して開きます。

```bash
python3 -m http.server 8000
```

ブラウザで `http://localhost:8000` を開いてください。

## ファイル構成

- `index.html` : UI レイアウト
- `styles.css` : ゲーム画面スタイル
- `app.js` : UI制御・イベント処理
- `sim.js` : ペナントロジック（日程・試合・育成・獲得・保存）
- `tests/test_sim.mjs` : シミュレーションテスト

## テスト

```bash
node --test tests/test_sim.mjs
```
